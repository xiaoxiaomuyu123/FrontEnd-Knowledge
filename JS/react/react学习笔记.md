# 1、创建项目    

react 官网使用 script 标签引入，仅用于学习调试。实际工作通过 react 脚手架创建项目进行开发部署    

- 1.安装脚手架 create react app     
  没有翻墙就用 `cnpm install -g create-React-app`
  已经翻墙就用 `npm install -g create-React-app`   
  
- 2.创建项目     
  `create-react-app (这里写自己的项目名称)`
